# API CRUD de Usuarios - .NET 8

Este proyecto es una API REST para gestionar usuarios, desarrollada con .NET 8.

## 🚀 Tecnologías
- .NET 8
- Entity Framework Core
- Swagger

## 📌 Funcionalidades
- Crear usuario
- Listar usuarios
- Obtener usuario por ID
- Actualizar usuario
- Eliminar usuario
